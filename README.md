Purpose of the project: The name of the app is social sparks and the aim of the app is to give social suggestions based on the time of day the user has entered. For example the user enters the time of the day such as morning afternoon or evening and an activty is suggested based on the time of day.
Features:user enters a time of day , The show suggestion button shows  activities for the user to do based on the time of day entered, The reset button clears the data so that more or another type of data can be inserted.
Input: The user enters a time of day for example morning,afternoon etc
Output: The app generates a response based on the time of day entered for etc send a good morning text to a family member.
Error Handling: The application looks for errors and mistakes for etc if the user enters the wrong time of day or leaves the edtTime field empty and clicks on the show suggetion the app generates a message saying " Invalid time of day".
